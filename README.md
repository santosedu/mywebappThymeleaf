# mywebappThymeleaf

This is a web tutorial used for studying some techs around Java/Spring Boot World.
Techs used:
- SpringBoot - 2.5.1 version;
- Spring Data JPA - same version;
- DevTools;
- Mysql database;
- Thymeleaf as main template engine frontend;
- HTML5;
- Bootstrap - 4.3.1 version;
